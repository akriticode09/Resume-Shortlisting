# Resume-Shortlisting
This project is a Machine Learning-based Resume Classification System that automatically categorizes resumes into different job domains based on their textual content. The objective is to streamline the recruitment process by helping recruiters quickly identify resumes relevant to specific job roles.
The project performs data preprocessing, text cleaning, feature extraction using Count Vectorization, and trains a Multinomial Naive Bayes Classifier to predict the category of a resume.

It demonstrates the complete machine learning workflow, including data preprocessing, exploratory data analysis, model training, and evaluation.
